## 通知模块

eg. 

- `GET /api/user/notices`

  获取所有通知 (接口对应的功能)

  权限级别：anon (接口的权限)

  | property  | desc                   |
  | --------- | ---------------------- |
  | page: int | 分页页码, default = 1  |
  | size: int | 分页大小, default = 10 |

  Response:

  ```json
  {
  	"code": 200,
  	"msg": "success",
  	"data": {
  		...
  	}
  }
  ```
